# Getting Started

ToolNexus started because I got tired of opening five different apps just to clean up one payload.

So this platform is pretty direct: open a tool, paste input, run, copy output. No account wall just to format JSON.

## Typical flow

1. Open the formatter or validator you need.
2. Paste input from logs, APIs, or local files.
3. Run the tool and sanity-check the result.
4. Move the output back into your actual workflow.

## One practical example

If you're debugging a flaky API response:

- run JSON Formatter first
- then JSON Validator
- then Text Diff to compare before/after payloads

That combo catches a lot of issues fast.
